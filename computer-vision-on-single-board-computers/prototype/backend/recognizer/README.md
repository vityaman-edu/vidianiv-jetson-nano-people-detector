## Build & Run

### Jupyter Notebooks
Install dependencies
```bash
pip3 install pytorch
pip3 install torchvision
pip3 install numpy
pip3 install opencv-python
```

Run notebooks in context of `notebook` directory
```bash
jupyter notebook notebook
```
